# 利用上の注意（OpenBeat Collector）

最終更新: 2026-06-16

本ツールは、報道・調査の効率化を目的に **公開情報のみ** を収集する補助ツールです。利用にあたっては以下を遵守してください。

## 1. 公開情報のみを対象とする

- 取得対象は、一般に公開されているRSS／記事ページ等の **公開情報** に限ります。
- 認証・有料・非公開領域への不正アクセスを行ってはなりません。ペイウォール等で取得できない場合は、本ツールの手動貼り付け（manual paste）機能を利用者の判断で用いてください。

## 2. 要約 + 出典リンクの原則

- 本ツールは、見出し・要約・出典リンクの保存を基本とし、**全文の再配布を目的とするものではありません**。
- 収集した本文を二次利用・公表する場合は、引用の範囲・著作権・出典表示について利用者が責任を負います。

## 3. robots.txt と利用規約の尊重

- 本ツールは収集にあたり **robots.txt を尊重** します。
- 各情報源の **利用規約（Terms of Use）の遵守は利用者の責任** です。規約上、機械的取得や再配布が制限される場合は、その情報源を登録・収集しないでください。

## 4. 個人データへの配慮

- 公開記事には個人データ（氏名・所属・発言等）が含まれる場合があります。収集・保存・利用にあたっては、適用される個人情報保護法（日本のAPPI、EU/英国GDPR）を遵守してください。

## 5. 責任分界

- 提供者（合同会社CODEAID）: プラットフォームの安全性・説明可能性・ログ・インシデント対応・セキュリティ・合理的なQA。
- 利用者: 入力の正確性、情報源の選定、利用規約の遵守、最終的な利用判断、専門的助言の取得。
