# プライバシーに関する説明（OpenBeat Collector）

最終更新: 2026-06-16

## 1. 本ツールの性質

OpenBeat Collector（以下「本ツール」）は、利用者のローカル端末上で動作する収集専用のデスクトップツール（Python / Flask / SQLite、Apache-2.0）です。本ツールは **127.0.0.1（ローカルホスト）にのみバインド**し、Webブラウザ画面は利用者本人の端末からのみ閲覧できます。

- 本ツール自体は、収集したデータを **外部サーバーへ送信しません**（オペレーターのサーバー、当社、第三者を含む）。
- AI処理・認証・課金機能は本リポジトリには含まれません。
- 情報源（RSS/サイト）への取得アクセスは、収集を実行するために必要な範囲で発生します。

## 2. 収集する情報

本ツールは、利用者が登録・選択した **公開情報源**（公開RSS／公開記事ページ等）のみを収集します。
収集される本文テキストには、**公開記事に登場する個人データ（氏名・所属・発言・連絡先等）が含まれる場合があります**。

収集データは利用者の端末上の **SQLiteデータベース（ローカルファイル）** に保存されます。

## 3. 保管・保持

- 保存先はローカル端末のみです（既定では実行ファイル近傍の `data/rwt.sqlite`）。
- **保持期間は利用者がコントロール**します。本ツールは自動削除を行いません。

## 4. 削除（消去）

GDPR第17条・個人情報保護法の趣旨に沿い、以下の削除手段を提供します。

- **情報源単位のカスケード削除**: 情報源を削除すると、その情報源から収集した本文行（raw_item）も連動して削除されます。
- **全件削除**: 収集済みデータをすべて消去するヘルパーを提供します。
  - Web UI: `/data/delete_all`（POST）
  - CLI: `python cli.py purge --yes`（特定情報源のみは `--source <id>`）

## 5. 管理者（データ管理者）の位置づけ

ローカルに保存されたデータについては、**本ツールを運用・利用する者（オペレーターまたは利用者）がデータ管理者（controller）**にあたります。本ツールの提供者（合同会社CODEAID）は、プラットフォームの安全性・説明可能性・ログ・インシデント対応・セキュリティ・合理的なQAについて責任を負いますが、収集対象の選定・入力の正確性・最終的な利用判断・専門的助言については利用者の責任となります。

## 6. 第三者提供・国際移転

本ツールはデータを外部送信しないため、本ツール単体による第三者提供・国際移転は発生しません。収集データを利用者が外部に持ち出す場合（エクスポート等）は、利用者の責任で適用法令を遵守してください。エクスポートJSONには個人データに関する注意書き（`privacy_note`）が含まれます。
